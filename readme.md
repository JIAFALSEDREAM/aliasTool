# 文件夹别名设置工具

利用 Windows 的 desktop.ini 机制，为指定文件夹设置自定义的显示名称（别名），实际路径保持不变。支持 UTF-8 输入，并根据系统 ANSI 代码页自动转换编码。

## 功能

- 修改文件夹在资源管理器中显示的名称，不影响实际路径
- 自动处理中文编码（GBK/UTF-8 转换）
- 交互式命令行界面，可重复设置
- 自动隐藏 desktop.ini 文件

## 使用方法

1. 从 Releases 页面下载 `aliasTool.exe`
2. 运行程序，按提示输入文件夹完整路径和想要显示的别名
3. 若已存在 desktop.ini 会询问是否覆盖
4. 刷新资源管理器即可看到新名称

## 原理

- 在目标文件夹下创建 `desktop.ini`，写入 `[.ShellClassInfo]` 和 `LocalizedResourceName=别名`
- 将 desktop.ini 设为隐藏 + 系统属性
- 给文件夹添加系统属性，使 Windows 读取该文件

## 注意

- 仅适用于 Windows
- 若改名后未立即刷新，可在文件夹属性 → 自定义 → 还原默认图标（点击应用）后刷新
- 删除别名只需删除 desktop.ini 文件（文件需要关闭隐藏系统文件选项）
